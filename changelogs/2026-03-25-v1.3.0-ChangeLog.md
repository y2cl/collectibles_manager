# v1.3.0 — 2026-03-25

PR #3 — Settings page, data source toggles, backup system, last-updated timestamps.

## Added

### Settings Page
- Settings page with three tabs: General, Data Sources, Debug
- General tab: duplicate strategy (merge/separate), paid merge strategy, default profile selection
- Data Sources tab: toggle each API source on/off (Scryfall, Pokémon TCG, Public Pokémon, eBay, SportsCardDatabase, SportsCardsPro)
- Debug tab: fallback cache stats and API diagnostics
- API config persisted to `api_config.json` via `save_api_config()`

### Backup System
- Auto-backup on collection save (optional, configurable)
- Configurable backup retention count
- Timestamped backup files created alongside source CSVs

### Last Updated Timestamps
- Pokémon cards sourced from local cache now show "Last updated" from TCGPlayer's `updatedAt` field
- MTG cards show last updated timestamp when available from Scryfall data
- `tcgplayer` JSON carried through local normalization pipeline

### Data Sources
- Disabled Scryfall/Pokémon API warnings shown when a source is toggled off but searched

## Fixed
- Fallback fuzzy name matching and wildcard suffix matching for set name lookups
