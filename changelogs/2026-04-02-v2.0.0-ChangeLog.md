# v2.0.0 — 2026-04-02

Complete rewrite — migrated from Streamlit single-file app to **FastAPI + React/TypeScript (Vite)**.

## Breaking Changes

- Application is no longer a Streamlit app. The legacy `collectiman.py` is preserved but no longer the active app.
- Data storage migrated from CSV files to **SQLite** (via SQLAlchemy ORM)
- All API endpoints are now REST (FastAPI), served at `http://localhost:8000`
- Frontend is a React SPA served at `http://localhost:5173` (or via the unified `npm run dev` command)

## Added

### Backend (FastAPI)
- `backend/` Python package with full FastAPI application
- SQLAlchemy ORM models: `CollectionCard`, `WatchlistItem`, `Owner`, `Profile`, `AppSettings`
- SQLite database with startup auto-migration for new columns
- CSV-to-SQLite migration script (`backend/migrations/csv_to_sqlite.py`)

### API Endpoints
- `GET/POST /api/collection` — list and add collection cards
- `PATCH/DELETE /api/collection/cards/{id}` — update and delete individual cards
- `POST /api/collection/cards/bulk-delete` — bulk delete
- `GET /api/search/mtg` — MTG card search via Scryfall
- `GET /api/search/pokemon` — Pokémon search via Pokémon TCG API / public fallback
- `GET /api/search/baseball` — Baseball search via eBay Finding API
- `GET/POST /api/owners` — owner management
- `GET/PUT /api/settings` — app settings
- `GET/PUT /api/settings/api-sources` — data source toggles
- `GET /api/sets` — sets browser (MTG + Pokémon from fallback CSVs)

### Frontend (React + TypeScript)
- Vite-based SPA with React Router
- Pages: Search, My Collection, Card Sets, Settings, Help
- TanStack React Query for all server state
- Zustand stores for owner selection, appearance, and settings persistence
- MTG search with double-faced card flip, foil/etched variant pills
- Pokémon search with variant price pills
- Baseball card search via eBay (manual add form)
- Collectibles manual add form with UPC barcode lookup
- Gallery view, Investment tab, Watchlist tab, Import/Export tab, Management tab

### Sets Browser
- MTG sets synced from Scryfall (`POST /api/sets/sync/mtg`)
- Pokémon sets synced from Pokémon TCG API (`POST /api/sets/sync/pokemon`)
- Per-set card cache download (`POST /api/sets/{code}/cache-cards`)
- Sortable columns including owned count
- Token/Memorabilia/Digital toggles for MTG

### Collection Page
- Sortable columns
- Bulk select, move, delete, and price refresh
- Move cards between owners/profiles

### Settings
- Appearance customization with color pickers, font selection, and live preview
- Presets: Dark Navy, Light, Forest, etc.
- General, Data Sources, Debug tabs

### Infrastructure
- Single `npm run dev` command runs both backend and frontend concurrently
- `README.md` with setup instructions
- `.nvmrc` for Node version pinning

## Removed
- Streamlit dependency (no longer required to run the app)
- SportsCardsPro.com — site blocks automated requests; removed from active sources
