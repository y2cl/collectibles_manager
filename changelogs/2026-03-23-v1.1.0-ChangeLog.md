# v1.1.0 — 2026-03-23

PR #1 — Help, Import improvements, Scryfall enrichment, fallback system overhaul.

## Added

### Import
- Improved CSV import with Scryfall enrichment (backfills missing prices and images on import)
- Template downloads for MTG and Pokémon CSV formats
- Better import error handling and reporting

### Fallback System
- Unified fallback manager: single module handles both MTG and Pokémon local cache reads/writes
- Offline-capable card lookup from local CSV caches

### Collection Management
- Owner folder structure with sanitized IDs
- Profile management per owner
- Collection zip export per owner
- Legacy data migration helper — copies default collection into a named owner folder
- Owner merge utility

### UI
- Tabs refactored: Collection / Gallery View / Investment / Watchlist / Import-Export / Collection Management
- Help view added with guidance on owners, profiles, import/export, and API settings

## Fixed
- Various simplification and unification of duplicated owner/profile path helpers
