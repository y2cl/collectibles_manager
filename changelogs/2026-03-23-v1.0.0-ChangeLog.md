# v1.0.0 — 2026-03-23

Initial release. Single-file Streamlit application (`collectiman.py`) for managing trading card collections.

## Added

### Core Application
- Streamlit-based single-page web app
- CSV-based data storage (no database required)
- Multi-owner support with per-owner folder structure and collection profiles
- Owner preferences persisted to `owner_prefs.json`

### Magic: The Gathering
- Card search via Scryfall API with local fallback cache (`mtgcards.csv`)
- Double-faced card support with flip button
- Foil / non-foil / etched variant pricing
- Add to collection or watchlist from search results
- Gallery view with card images, price pills per variant

### Collection Management
- Collection tab with sortable table view
- Gallery view with configurable cards-per-row and image width
- Investment tab (value tracking)
- Watchlist tab
- Import/Export tab — CSV import/export with duplicate merge strategy
- Collection Management tab — owner/profile creation, deletion, merge, zip export

### Data & Fallback
- Scryfall enrichment on import (backfills missing prices/images)
- Local fallback cache for offline use (`fallback_data/MTG/mtgcards.csv`)
- Set metadata cache (`mtgsets.csv`, `pokemonsets.csv`)
- API config persisted to `api_config.json`

### Help
- In-app help page explaining owners, profiles, import/export, and API settings

### Settings
- Data Sources tab — toggle Scryfall, Pokémon TCG, eBay, SportsCardDatabase on/off
- Debug tab — fallback cache stats, API health checks
