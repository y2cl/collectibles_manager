# v2.2.0 — 2026-04-03

Sports Cards feature branch — major additions for sports card collectors.

## Added

### Sports Cards — Search & Add
- Renamed "Baseball Cards" to "Sports Cards" covering all major sports
- Per-sport selector: Baseball, Football, Basketball, Hockey, Soccer, Other
- eBay Finding API searches by sport using sport-specific category IDs
- Sports card add form with autocomplete for Player, Set, Insert, Grading Company

### Sports Cards — Grading & Certification
- Grading Company, Grade, and Cert. Number fields
- Cert. lookup links: PSA, Beckett (BGS), SGC, CGC — opens grading company cert lookup in new tab
- PSA cert auto-fill — enter a PSA cert number and click Lookup to auto-fill Player, Year, Set, Card #, and Grade from the PSA public API
- Serial # field for serialized/print-run cards (e.g. 23/99), separate from Cert. Number
- RC (Rookie Card) checkbox
- Signed checkbox

### Sports Cards — Sets Browser
- Sports Cards tab added to the Sets page
- User-defined set catalog: Add, Edit, and Delete set entries
- Set fields: Sport, Set Name, Insert, Year, # Cards in Set, Link (URL), Notes
- Aggregated stats per set: # Owned, Paid, Value (computed from collection)
- Expandable insert sub-rows per set
- "View Cards" button deep-links to My Collection filtered by that set/insert
- Set and insert names from catalog appear in add-form autocomplete

### My Collection — Filters
- Player/Card Name, Set, Insert/Variant, and Year filter inputs for all game views
- Filters replace the old single "Search cards" box
- Sports Cards view deep-link from Sets page pre-fills Set and Insert filters

### My Collection — Edit Modal
- Edit modal for Sports Cards and Collectibles cards
- Editable fields: all identity and grading fields, image URL, price paid, value
- Cert. Number lookup link shown in edit modal when grading company and cert are filled

### Image Search
- Switched from Google Custom Search (deprecated Jan 2026) and Bing Search V7 (retired Aug 2025) to **SerpAPI** (`engine=google_images`, 250 free searches/month)

### Settings
- Change Log tab — lists all version changelogs, click any entry to read the full notes
- Changelog file structure: `changelogs/YYYY-MM-DD-vX.Y.Z-ChangeLog.md`

## Removed
- SportsCardsPro.com — requires paid subscription; removed from data sources, settings, and all backend/frontend code
