# v1.2.0 — 2026-03-24

PR #2 — Pokémon TCG support, variant pricing, deduplication improvements.

## Added

### Pokémon TCG
- Full Pokémon card search via Pokémon TCG API (`api.pokemontcg.io`)
- Public (no-key) endpoint fallback for Pokémon search
- Local fallback cache for Pokémon cards (`pokemoncards.csv`)
- Set metadata stored to `pokemonsets.csv` on search

### Variants
- Pokémon variant pricing: Normal, Holofoil, Reverse Holo, 1st Edition, and custom variants
- TCGPlayer-style price key mapping to friendly variant names
- Custom variant text input with manual price entry
- Variant shown in collection table and stored per card

### Collection
- Duplicate merge strategy applied to Pokémon cards on add (merge quantities vs. separate entries)
- Watchlist deduplication with same merge logic
- Pokémon-specific collection export to `pokemon_collection.csv`

### UI
- Pokémon search results rendered with variant price pills (same UI pattern as MTG)
- Cards-per-row and image width sliders for Pokémon gallery view
